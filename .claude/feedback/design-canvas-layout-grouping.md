# 设计稿画布排版分组策略

**问题描述**：Pencil 设计稿的排版布局全部纵向堆叠，不便于查阅和管理。用户要求按功能分类或按相同页面类型分组排版。

**触发场景**：design-maker 完成设计稿后，用户审查画布排版时提出布局意见。

**教训/建议**：在 Pencil 中排版设计稿时，应按以下分组方式布局：

- 欢迎/引导类放一排
- 主应用界面放一排
- 面板/弹窗类放一排
- 状态细节放一排

按类别分组排版，便于用户快速定位和理解设计结构。
